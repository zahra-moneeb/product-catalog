# Product Catalog

A simple React project to display products in a catalog with a clean and responsive UI.

## Live Demo
[View Live](https://zahra-moneeb.github.io/product-catalog/)

## Features
- Display products in a grid layout
- Responsive design using CSS/Bootstrap
- Product details page
- Client-side routing with React Router

## Installation
git clone https://github.com/zahra-moneeb/product-catalog.git
cd product-catalog
npm install
npm run dev
